Tributary

Tributary started with the idea of making responsive code easier to share. We started with Gabriel Florit's implementation of Bret Victor's talk. The use of gists to drive the code in the view comes from Mike Bostock's bl.ocks.org. Tributary works by evaluating your code, providing a few things such as an svg element and a few libraries for the code to use.

Features

Tributary comes with jQuery, d3, underscore and backbone available in the code editor. The editor is powered by CodeMirror, jQuery UI and ColorPicker
